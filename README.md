<h2>🛠️ Installation Steps:</h2>

<p>1. Récupérer le repo</p>

```
git clone https://github.com/nsWilfried/Angular-jwt-server
```

<p>2. Installer les dépendances du serveur</p>

```
npm install 
```

<p>3. Lancer le serveur</p>

```
npm run dev
```
